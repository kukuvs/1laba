1. x = c = a;: Присваивание значения a переменным c и x. Значения после этой строки: a = a, c = a, x = a.

2. a += b;: Увеличение a на значение b. Значение после этой строки: a = a + b.

3. x *= b + a;: Умножение x на b + a. Значение после этой строки: x = x * (b + a).

4 . b += a--;: Увеличение b на значение a, затем уменьшение a на 1. Значение после этой строки: a = a - 1, b = b + a.

5. x -= ++c;: Уменьшение x на значение c после увеличения c на 1. Значение после этой строки: c = c + 1, x = x - c.

6. c = a / b;: Присваивание результат деления a на b переменной c.

7. c = a % b;: Присваивание остатка от деления a на b переменной c.

8. y += (a + 1) / a++;: Прибавление значения к y, где a сначала используется, затем увеличивается на 1.

9. b = 3 * (y -= 0.6) + 2 * b + 1;: Вычисление значения y, уменьшение на 0.6, затем используется для вычисления b.

10. z = a / 2;: Присваивание результат деления a на 2 переменной z.

11. z = (double)a / 2;: Присваивание результата деления a на 2 в форме double переменной z.

12. y = (x = 5.7) / 2;: Присваивание значения x, затем результат деления x на 2 переменной y.

13. y = (int)x / 2;: Присваивание целой части значения x, деленной на 2, переменной y.
